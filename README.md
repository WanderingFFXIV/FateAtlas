An Addon to List Fates by Level and Area based upon currently equipped class.
Also includes Local Area Fate information and tracking.

## Data sources

FATE leveling route information was compiled from publicly available
Final Fantasy XIV community resources and independently organized for
FATE Atlas.

Community references include:

- Final Fantasy XIV Online Wiki
- Dalamud API documentation
- Lumina and FFXIVClientStructs documentation

No game artwork, icons, audio, or other Square Enix assets are distributed
with this project.
